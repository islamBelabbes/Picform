## Installation

1. Clone the repository:

```bash

git clone https://github.com/islamBelabbes/Picform.git

```

2. Navigate to the project directory:

```bash

cd Picform

```

3. Install the required dependencies:

```bash

npm install

```

4. Start the development server:

```bash

npm run dev

```
